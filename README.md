  - ARRL-10
  - ARRL-160
  - ARRL-DX-CW
  - ARRL-DX-SSB
  - ARRL-RTTY
  - ARRL-SS-CW
  - ARRL-SS-SSB
  - ARRL-VHF-JAN
  - ARRL-VHF-JUN
  - ARRL-VHF-SEP
  - CA-QSO-PARTY
  - CQ-WPX-SSB
  - CQ-WPX-RTTY
  - CQ-WPX-CW
  - CQ-WW-RTTY
  - CQ-WW-CW
  - CQ-WW-SSB
  - NAQP-CW 
  - NAQP-RTTY
  - NAQP-SSB
  - NA-SPRINT-CW
  - NA-SPRINT-SSB

1. QSO Count
  - By Band
  - By Mode
  - By Country (DX)
  - By US State (WAS)
  - By Grid
  - By Band Mode
1. QSO Rates
  - Total
  - By Day
  - By Hour
1. Mutliplers
  - defined per Contest
